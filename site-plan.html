Site Plan for My Personal Portfolio Website

Purpose

This website will showcase my skills, experience, and personal projects as a developer. It will serve as a professional portfolio to highlight my abilities, code samples, and achievements.

Audience

Prospective employers, clients, or collaborators looking to view my work and professional background.

Dynamic Elements

  • Contact form with validation and submission handling using JavaScript.
  • Projects gallery with dynamic content using JavaScript arrays.
  • Scroll animation effect triggered as the user navigates through the page.

Logo

Simple, clean logo created on Canva, featuring initials or a symbol representing development.

Colors and Fonts

Primary Colors: Blue (#3b5998), White (#FFFFFF), Light Grey (#f4f4f4).

Fonts: 'Roboto' for body text, 'Poppins' for headings.

Content

  • Home page: Introduction, skills, and featured projects.
  • Child page(s): Detailed project descriptions and technical case studies.
  • Contact page: Form for inquiries.
